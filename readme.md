
# Casino game task

This is my coding example. Total time of development is circa 8 hours.

## Prerequisites

* Java 21
* Maven 3.8.7
## Deployment and running

The code is using `Makefile` for easier builds and package exporting.

#### Build

```bash
  make build
```
#### Make runable jar file

```bash
  make package
```
#### Run program

```bash
  make run
```
